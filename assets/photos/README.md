# Фото для Quantum Elite School

Сайт уже готов работать без фотографий: пока изображения не добавлены, он показывает красивые плейсхолдеры с подсказками.

Вы можете добавить фото двумя способами:

1. Через скрытую админ-панель `admin/login.html`
2. Или вручную положить изображения в папку `assets/photos/` и вписать пути

Рекомендуемые файлы:

- `assets/photos/classroom-kids.jpg` — дети в классе
- `assets/photos/teacher-explains.jpg` — учитель объясняет
- `assets/photos/school-interior.jpg` — школа внутри
- `assets/photos/school-exterior.jpg` — школа снаружи
- `assets/photos/happy-students.jpg` — дети улыбаются
- `assets/photos/english-class.jpg` — урок английского
- `assets/photos/clubs.jpg` — кружки
- `assets/photos/school-life.jpg` — атмосфера школы
- `assets/photos/teacher-1.jpg` — преподаватель 1
- `assets/photos/teacher-2.jpg` — преподаватель 2
- `assets/photos/teacher-3.jpg` — преподаватель 3
- `assets/photos/review-parent.jpg` — фото к отзыву, если нужно

По стилю выбирайте:

- светлые кадры
- чистый интерьер
- живые эмоции
- счастливые дети

Не используйте:

- тёмные фото
- старые снимки
- безэмоциональные стоковые изображения
