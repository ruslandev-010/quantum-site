Как работает Telegram интеграция

Для автоотправки заявок нужен Bot Token вашего Telegram-бота и Chat ID того чата, куда должны приходить заявки. Это может быть ваш личный Telegram, группа администраторов или канал через привязанного бота.

Быстрый порядок такой: 1) создать бота через BotFather, 2) получить от него токен, 3) написать вашему боту хотя бы одно сообщение, 4) узнать Chat ID того чата, куда бот должен слать заявки.

Важно: текущая версия сайта статическая, поэтому токен хранится в браузере. Для продакшена лучше перенести отправку на сервер.
